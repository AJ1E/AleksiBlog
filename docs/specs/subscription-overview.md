# 订阅概览改造说明

## 目标

将首页订阅概览和登录后的订阅管理抽屉改为展示四项真实订阅，保留当前简洁、安静的卡片风格。

## 范围

- ChatGPT Business Team、阿里云服务器、Microsoft Azure、阿里云域名。
- 首页显示四个独立订阅条目、月均费用和最近续费提醒。
- 详情抽屉显示美元与人民币近似金额、续费日期、剩余天数、计费周期、月均折算和续费链接。
- 分类字段保留给后续扩展，但本阶段不显示分类占比和分类筛选。

## 不做的事情

- 不改独立 `/subscriptions` 页面的视觉结构。
- 不新增订阅计费 API、数据库或外部运行时服务。
- 不保存银行卡、密码、Token 或其他支付凭据。

## 验收标准

- 订阅数量为 `4/4`，并按下一次续费日期动态排序。
- 美元和人民币金额均来自同一份项目汇率快照，并标记为近似值。
- `pnpm build` 通过，移动端和深色模式无明显溢出或低对比度。
