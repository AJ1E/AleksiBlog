# 项目卡片动效与中文化

## 目标

- 让项目页的每张项目卡在悬停或键盘聚焦时拥有与项目主题相符的轻量 SVG 动效。
- 将现有三个示例项目改为中文内容，并新增“个人博客与工作台”项目。

## 范围

- 修改项目内容集合中的四个项目条目。
- 扩展 `ProjectVisual.astro` 的 SVG 场景。
- 在现有项目卡片 hover CSS 中加入对应动画。

## 不做的事

- 不引入 GIF、JavaScript 动画库或第三方依赖。
- 不改变项目页的卡片布局、路由结构或部署配置。

## 验收

- 四张项目卡的标题、简介、分类、标签和链接文案均为中文。
- 四种封面在悬停时分别呈现仪表盘数据、内容流转、服务器心跳、博客阅读流的主题动作。
- 每个项目详情页标题区第一行都说明封面缩写、英文全称与中文含义。
- 键盘聚焦可触发动效；`prefers-reduced-motion` 下不播放动画。
- `pnpm build` 通过，并在本地检查桌面与移动端项目页。
